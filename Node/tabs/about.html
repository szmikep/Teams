Thank You for using Microsoft Teams!

This is a sample app to showcase many of the awesome capabilities in Microsoft Teams

Video recordings of how this app was built are at: aka.ms/teamsBuildVideos

Documentation on how to build apps for Microsoft Teams can be found at: aka.ms/TeamsDocs
